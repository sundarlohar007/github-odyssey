## ✨ Features

### ⚔️ RPG-Progression System
- **XP & Levels**: Every commit, PR, and review earns you XP
- **10 Tiers**: From Noob to Divine, climb the ranks
- **Streak Bonuses**: Multipliers for consistent contributors

### 🏅 Achievement System
- **20+ Badges**: Common, Rare, Epic, and Legendary achievements
- **Shareable Badges**: Generate SVG badges for your profile
- **XP Bonuses**: Earn bonus XP for unlocking achievements

### 📊 Player Cards
- **Customizable Cards**: Beautiful SVG cards showing your stats
- **Profile Ready**: Add to your GitHub README
- **Real-time Data**: Always up-to-date with your GitHub activity

### ⚔️ Battle Mode
- **1v1 Duels**: Compare stats with any developer
- **Power Rankings**: See who comes out on top

### 🌍 Global Leaderboard
- **Top Developers**: See who's leading the pack
- **Weekly Updates**: Fresh rankings regularly

## 🚀 Quick Start

### Installation

```bash
# Using npm
npm install -g github-odyssey

# Using npx (no install needed!)
npx github-odyssey --help
```

### Setup

```bash
# Initialize with your GitHub token
odyssey setup --token YOUR_GITHUB_TOKEN
```

Get your token at: https://github.com/settings/tokens

### Basic Commands

```bash
# View your profile
odyssey profile

# Sync your data
odyssey sync

# Check achievements
odyssey achievements

# Generate a player card
odyssey card

# Battle a friend
odyssey battle yourfriend

# View leaderboard
odyssey leaderboard
```

## 🎮 Commands

| Command | Description |
|---------|-------------|
| `odyssey profile [user]` | View your or another user's profile |
| `odyssey stats [user]` | View detailed statistics |
| `odyssey achievements` | View all achievements |
| `odyssey leaderboard` | View global rankings |
| `odyssey battle <user>` | Battle another developer |
| `odyssey sync` | Sync your GitHub data |
| `odyssey badge [id]` | Generate achievement badge |
| `odyssey card [user]` | Generate player card |
| `odyssey setup` | Configure GitHub token |
